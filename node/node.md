# node
확장자 js, mjs, jsx등을 실행
node 파일명.확장자 : 확장자는 생략가능

## node 설치 및 삭제
1. 설치: npm i 패키지명(모듈명) / npm install 패키지명(모듈명)
2. 셋팅: npm init -y : 기본적인 셋팅은 모두 해달라. 
3. 삭제: npm uninstall 패키지명(모듈명)
