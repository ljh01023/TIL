# 애플리케이션 (application)

응용 프로그램.
다양한 프로그래밍 언어로 애플리케이션을 만들 수 있다.
ex) 웹, 웹 서버, 모바일(리액트 네이티브), 데스크톱 애플리케이션(일렉트론) 등

# node

확장자 js, mjs, jsx등을 실행
node 파일명.확장자 : 확장자는 생략가능

## node 설치 및 삭제

1. 설치: npm i 패키지명(모듈명) / npm install 패키지명(모듈명)
2. 셋팅: npm init -y : 기본적인 셋팅은 모두 해달라.
3. 삭제: npm uninstall 패키지명(모듈명)
